# My homebrew-tap

> Unified tap to distribute some apps I use on Homebrew.

## Usage

Any included software can be installed directly using either of these commands:

```sh
# Formulae
$ brew install moritzleidinger/tap/<formula>

# Casks
$ brew install --cask moritzleidinger/tap/<cask>
```

Alternatively, you can "tap" into the repository, basically adding it as installation source, and discard the prefixes when installing:

```sh
brew tap moritzleidinger/tap
```

### Formulae

```sh
brew install <formula>
```

### Casks

```sh
brew install --cask <cask>
```
